# Harry Potter Game Project

This repository contains my final project for **COGS 18 (Introduction to Python)** which is a Harry Potter inspired game.

---

## Repository Contents

- **ProjectNotebook.ipynb**  
  Main project notebook (analysis, results, and visuals).

- **functions.py**  
  Helper functions used by the notebook.

- **test_quest_functions.py**  
  Test script for validating the helper functions.
