# Harry Potter Game Project

This repository contains my final project for **COGS 18 (Introduction to Python)** which is a Harry Potter inspired game.

---

## Repository Contents

- **ProjectNotebook.ipynb**  
  Main project notebook (analysis, results, and visuals).

- **functions.py**  
  Helper functions used by the notebook.

- **test_quest_functions.py**  
  Test script for validating the helper functions.

---

## Getting Started

### Clone the repository

```bash
git clone https://github.com/shenova/cogs18project.git
cd cogs18project
```
### Running the Project

Open the Jupyter Notebook:
jupyter notebook ProjectNotebook.ipynb

(Optional) Run tests:
python test_quest_functions.py

## Project Summary

Goal:
Build an interactive, text-based game using Python that applies core programming concepts taught in COGS 18.

Approach:
The game logic is modularized into helper functions, tested for correctness, and executed through a Jupyter Notebook to demonstrate gameplay flow.

Results:
Successfully created a functional Python game with reusable functions and test coverage.
